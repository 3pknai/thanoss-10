Чему можно научиться в Educatics?

Здесь вы можете научиться основам программирования, если пройдёте курс из уроков и будете добросовестно выполнять тесты. Вы обучитесь вводу и выводу, простым типам данных, условиям и циклам на языке программирования python. Почему именно python? Одним из главных преимуществ python является простота его освоения. Чтобы изучать язык, вам не нужно знать программирование: достаточно понимания основ математики. Этот язык программирования подходит для возрастной категории 5-6 классов. Так что, если хотите научиться программировать - вы пришли по адресу)

Как начать программировать на python?

Это очень просто. Следуте инструкции:

  1. Зайдите на официальный сайт python.org и откроете раздел Downloads. Скачайте самую последнюю версию python для вашей операционной системы.
  2. Чтобы запустить встроенную IDE, введите во встроенном поисковом устройстве компьютера "IDLE" и откройте соответствующий файл. Вы готовы к написанию программы!
  3. Чтобы запустить вашу программу, найдите вкладку Run -> Run Module ИЛИ нажмите клавишу F5.
